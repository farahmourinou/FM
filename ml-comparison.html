Supervised Learning Comparative Analysis

Python Scikit-Learn Neural Networks (MLP) SVM Kernel Ridge

🔬 Research Scope

This project evaluates the performance of multiple supervised learning architectures. The analysis covers Multi-class Classification using the Iris dataset and Non-linear Regression using New Taipei City real estate data, with a specific focus on how Multi-Layer Perceptrons (MLP) compare to traditional statistical methods.

🏷️ Classification Results (Iris)

Algorithm Accuracy Time (s)
Decision Tree97.78%0.0009
SVM (Linear)97.78%0.0011
MLP Classifier95.56%0.1240
KNN95.56%0.0023

Insight: While MLP is highly flexible, Decision Trees achieved the same accuracy with significantly lower computational overhead for this dataset.

📈 Regression Results (Housing)

Method MSE R² Score
Kernel Ridge71.650.572
OMP72.700.565
MLP Regressor73.120.563
LASSO73.680.559

Insight: Kernel Ridge (KRR) provided the best fit, though the MLP Regressor showed competitive generalization capabilities on non-linear price patterns.

🧮 Applied Methodologies

Artificial Neural Networks (MLP)

Implemented using hidden layers and backpropagation to capture complex, multi-dimensional relationships.

Kernel Regression

Applied RBF and Polynomial kernels to handle non-linearity without explicit feature mapping.

Iterative Feature Selection

Utilized Orthogonal Matching Pursuit (OMP) for sparse representation and dimensionality reduction.

⚙️ Implementation Details

  • Standardization: StandardScaler applied to all continuous features.
  • MLP Config: 2 hidden layers (100, 50), ReLU activation, Adam optimizer.
  • Validation: 70/30 Train-Test split with controlled random seeding.
  • Metrics: Mean Squared Error (MSE) and R-squared for regression.

Conclusion

The results demonstrate that the choice of algorithm depends heavily on data scale and dimensionality. While MLP excels in complex pattern recognition, Kernel Ridge and Decision Trees often provide superior efficiency and interpretability for smaller, structured datasets.
