Facial Emotion Recognition

TensorFlow Keras CNN Adam Optimizer

🌟 Project Overview

Built an automated classification system designed to identify human emotions from facial images. This technology aims to assist individuals who struggle with interpreting non-verbal social cues or facial expressions.

The model identifies five key emotion categories: Sad, Happy, Disgusted, Angry, and Fearful.

🏗️ Architecture Details

  • Framework: Built using TensorFlow and Keras.
  • Layers: Multiple Convolutional blocks for feature map generation + ReLU activations.
  • Classification: Dense (Fully Connected) layers with Softmax for class probabilities.

⚙️ Training Specs

  • Loss Function: Categorical Cross-Entropy for multi-class labels.
  • Optimizer: **Adam** for adaptive learning rate adjustment.
  • Downsampling: Max-Pooling layers used to improve computational efficiency.
