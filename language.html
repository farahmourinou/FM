Semi-Supervised Translation System

Scikit-Learn NLP MarianMT Hugging Face Self-Training

🌐 Project Mission

Development of an intelligent system capable of automatically identifying the language of a text (English or French) and providing an instant neural translation. This system utilizes Semi-Supervised Learning to leverage unlabeled data, maximizing accuracy despite limited human-annotated samples.

🔍 Step 1: Language Detection

An MLP (Multi-Layer Perceptron) base classifier is enhanced via Self-Training:

  • TF-IDF Vectorization: Numerical mapping of text relevance.
  • Confidence Threshold: Only predictions above 75% are accepted as pseudo-labels.
  • Robustness: High-density manifolds are captured to refine decision boundaries.

🚀 Step 2: Neural Translation

Integration of MarianMT from the Hugging Face Transformers library:

  • Bi-directional: Seamlessly handles English ➔ French and French ➔ English.
  • Model: Optimized Helsinki-NLP opus models for performance.
  • Output: High-fidelity translation maintaining grammatical context.
